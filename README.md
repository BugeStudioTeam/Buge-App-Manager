
1. **Authorization** – Uses Shizuku to gain privileged access for app and permission management
2. **App Overview** – Browse all installed apps in a clean, searchable list
3. **Permission Management** – Select any app to view and modify its permissions by category
4. **Privacy Control** – Review granted vs. denied permissions to maintain privacy

## 📋 Requirements

| Requirement | Minimum Version |
|-------------|-----------------|
| Android OS | 8.0 (API 26) or higher |
| Shizuku | Installed and authorized |

> **Note**: [Shizuku](https://shizuku.rikka.app/) is required for advanced permission management features.
